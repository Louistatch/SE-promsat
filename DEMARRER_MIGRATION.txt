================================================================================
                MIGRATION VERS NEON + FIREBASE - DEMARRAGE RAPIDE
================================================================================

ETAPE 1: PREPARATION (5 min)
================================================================================

1. Lire le guide complet:
   GUIDE_MIGRATION_COMPLET.md

2. Creer un compte Neon:
   https://neon.tech

3. Creer un projet Firebase:
   https://console.firebase.google.com


ETAPE 2: CONFIGURATION (10 min)
================================================================================

1. Executer le script de configuration:
   python setup_neon_firebase.py

2. Suivre les instructions et entrer:
   - Chaine de connexion Neon
   - Identifiants Firebase

3. Telecharger firebase-credentials.json depuis Firebase Console


ETAPE 3: MIGRATION (10 min)
================================================================================

1. Sauvegarder les donnees actuelles:
   python manage.py dumpdata > backup.json

2. Executer la migration:
   python migrer_vers_neon.py

3. Verifier les donnees:
   python verifier_donnees.py


ETAPE 4: TESTS (5 min)
================================================================================

1. Demarrer l'application:
   python manage.py runserver

2. Tester la connexion:
   http://localhost:8000/accounts/login-firebase/

3. Creer un compte et se connecter
